Per poter installare Squid-gui da sorgente bisogna 
scaricare il programma all'indirizzo:



Una volta scaricato aprire il terminale e dicitare i seguenti comandi

tar -xvzf squid-gui-0.2.tar.gz

cd squid-gui-0.2

Prima di installare Squid-Gui installare 
squid e dansguardian

Dipendenze:
qtBase-qt5
qtbase-qt5-gui
polkit-qt5

Compilare con le Qt in versione 5
eseguire il seguente comando

cmake -DMAKE_INSTALL_PREFIX=/usr -DWANT_QT5=ON 

make-j5
sudo make install
